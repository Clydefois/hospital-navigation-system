# Hospital Background Image

To add your hospital background image:

1. Place your hospital image as `hospital-background.jpg` in this `/public` folder
2. Recommended image dimensions: 1920x1080 or higher for best quality
3. The image will be used as the hero background on the landing page

## Alternative image names supported:

- hospital-background.jpg (primary)
- hospital-background.png
- hospital-hero.jpg
- medical-center.jpg

If no image is provided, the page will display a blue gradient background as fallback.
